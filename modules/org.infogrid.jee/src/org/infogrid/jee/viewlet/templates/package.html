A templating mechanism for JEE Viewlets.

All output by InfoGrid Viewlets is deposited in an instance of class {@link org.infogrid.jee.viewlet.template.StructuredResponse StructuredResponse}, which is composed of one or more named {@link org.infogrid.jee.viewlet.template.StructuredResponseSection StructuredResponseSections}.

{@link org.infogrid.jee.viewlet.template.StructuredResponseTemplate StructuredResponseTemplate} is then used to assemble some or all of the StructuredResponseSections into the output stream.
