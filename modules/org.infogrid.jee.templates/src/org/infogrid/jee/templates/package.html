A templating mechanism for InfoGrid JEE applications.

All output is deposited in an instance of class {@link org.infogrid.jee.template.StructuredResponse StructuredResponse}, which is composed of one or more named {@link org.infogrid.jee.template.StructuredResponseSection StructuredResponseSections}.

{@link org.infogrid.jee.template.StructuredResponseTemplate StructuredResponseTemplate} is then used to assemble some or all of the StructuredResponseSections into the output stream.
